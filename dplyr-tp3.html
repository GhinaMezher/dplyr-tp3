It is too long to type all variables concerning the Curricular units. By using the across function we are able to include more than one variable without typing them out one by one by sifting through all the variables of the database. Starts_with helps us give a selection to the selection of variables by across, here the condition is starting with ‘Curricular units’.

We could also attribute the code to a variable, mean_val and then do print(mean_val) in order to reuse it in the rest of our code without typing the question 3 code all over again.
